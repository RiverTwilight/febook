# 网格 & 弹性布局

## 弹性布局

熟练记忆**容器属性**和**项目属性**是掌握弹性布局的关键。

### 容器属性

-   flex-direction: 规定布局方向
-   flex-wrap
-   flex-flow
-   justify-content
-   align-items
-   align-content

### 项目属性

-   order
-   flex-grow
-   flex-shrink
-   flex-basis
-   flex
